  3. 软件工程

    给定成本、进度的前提下，开发出具有有效性，可维护，可追踪，可复用，可移植的软件产品。

2. 软件的生命周期(软件开发流程)

俗称 三个周期八个阶段

  1. 软件定义期

    1. 可行性研究阶段

      技术，人力，资金，设备，时间，回报率，政策，风俗。。。

    2. 需求分析阶段、

      分为功能性需求分析和非功能性需求分析

      非功能性需求对功能性需求的实现方式

  2. 软件的开发期

    1. 概要设计阶段——架构师

      子模型，系统，技术选型，数据库设计

    2. 详细设计阶段——模块负责人

      页面，主题内容，属性，功能、

    3. 编码实现阶段

      U I设计——网页的效果图

      前端工程师——将网页效果转换为 html，css，js格式的文件

      后端工程师——为前端提供项目的数据

    4. 测试阶段——测试工程师

  3. 软件维护期

    1. 部署阶段——运维工程师

      将软件部署到服务器

    2. 维护阶段

4. 项目中的服务器

​ 硬件：就是一台功能强大的计算机

​ 软件：可以提供各种服务，例如WEB服务，数据库服务，游戏服务，文件服务

访问服务器

​ 域名/IP：找服务器

​ 端口：服务器下的每一个服务器都有对应的端口

​ 协议：客户端要使用服务需要对应的协议
